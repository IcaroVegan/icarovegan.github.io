How to play Ajeita Drika

The main objective of this game is to preserve the building as well as you can fixing every damage that Ralph has made to it.

Game duration is 120 seconds and at the end of this time you should have at least 10 points left.

The game could end before 120 seconds if the total points is equal to 0

Commands :

  • move left -> left arrow
  • move right -> right arrow
  • move up -> up arrow
  • move down -> down arrow
  • fix the window ->Ctrl left
  • pause -> p

You can't fix windows which has both cristals broken, only when one of them is unbroken
